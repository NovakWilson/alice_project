Алиса понимает корректно введенные фразы, описанные в списке команд.

функции:

1) def get_city(req) - проверяет на существование введенный город.

2) def get_first_name(req) - проверяет на существование введенное имя.

3) get_coordinates(city_name) - возвращает координаты введенного адреса в формате широты и долготы.

4) def get_distance(p1, p2) - возвращает расстояние между двумя введенными населенными пунктами/объектами.

5) def get_country(city) - возвращает название страны введенного города.

6) def translate(text, lang='en') - возвращает перевод текста с любого языка на язык, выбранный пользователем. 

Допольнительной опцией является выдача рекомендации по одежде с учетом прогноза погоды в заданном районе.
